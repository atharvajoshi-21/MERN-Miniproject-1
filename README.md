## Node.js Social Mini Project ##
This is a mini full-stack backend project built using Node.js and Express.  
It allows users to register, login, upload profile avatars, create posts, like/unlike posts and interact in a shared feed.  
The project uses MongoDB for database, JWT authentication for security, and Multer for image upload handling.

## Features
- User Registration & Login (JWT Authentication)
- Profile Page with user details
- Upload profile picture using Multer
- Create, Edit, and View Posts
- Like/Unlike Posts
- Add Comments on feed posts
- MongoDB & Mongoose Database Connection
- EJS Templating Engine

## Tech Stack
- Node.js
- Express.js
- MongoDB & Mongoose
- JWT Authentication
- Multer (Image Upload)
- Bcrypt (Password Hashing)
- EJS for UI Rendering
